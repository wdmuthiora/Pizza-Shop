# Pizza Shop Website.

## Demo.
Here is a working live demo :  https://wdmuthiora.github.io/Pizza-Shop/


## Site.
This is a site for my Pizza shop. It is designed to allow a customer to choose the size of pizza, what kind of crust and topping(s) they want on their pizza, then place an order.
